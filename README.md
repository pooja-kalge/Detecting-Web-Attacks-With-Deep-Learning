# Detecting Web Attacks with End-to-End Deep Learning

## Overview

This project is a web attack detection system developed using Deep Learning techniques. The system analyzes web traffic data and predicts whether a request is malicious or legitimate.

## Features

* Detects web attacks using Deep Learning.
* User registration and login system.
* Upload and analyze web traffic data.
* Predict attack types from input data.
* Visualization of prediction results.

## Technologies Used

* Python
* Django
* TensorFlow / Keras
* NumPy
* Pandas
* Scikit-learn
* HTML
* CSS

## Installation

1. Clone the repository:

```bash
git clone https://github.com/pooja-kalge/Detecting-Web-Attacks-With-Deep-Learning.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python manage.py runserver
```

4. Open the browser and visit:

```text
http://127.0.0.1:8000/
```

## Dataset

The project uses web traffic data for training and testing attack detection models.
